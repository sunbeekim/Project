Movie Info Project

영화 정보 검색과 커뮤니티 기능을 제공하는 웹 애플리케이션입니다. 한국영화데이터베이스(KMDB)와 영화진흥위원회(KOFIC) Open API를 활용하여 다양한 영화 정보와 순위를 제공합니다.

영화 검색 상영 영화 순위

주요 기능

영화 검색

  • 제목, 감독, 배우 검색
  • 연도별, 국가별 필터링
  • 장르 분포 시각화
  • 주요 배우 출연 현황
  • 주요 감독 작품 수
  • 주요 장르 분포

영화 순위

  • 일간 순위
  • 관람객 수 통계

커뮤니티

  • 게시글 CRUD
  • 조회수 관리
  • 사용자 권한 관리
  • 반응형 UI

회원 관리

  • 세션 기반 인증
  • 회원가입/로그인
  • API 인증 검증

기술 스택

Frontend

  • Google Charts
  • AJAX
  • jQuery
  • Bootstrap
  • Thymeleaf
  • REST API

Backend

  • Spring Boot 3.3.7
  • Spring Framework
  • Spring Mybatis
  • MySQL
  • RESTful API

시스템 구성

아키텍처

  • Frontend: Thymeleaf 기반 템플릿 엔진
  • Backend: 계층형 아키텍처 (Controller-Service-Repository)
  • Database: MySQL (게시판, 회원정보)
